Support

MyDrip サポート

MyDrip の使い方、問い合わせ先、Issue 受付方法をまとめたサポートページです。Chrome Web Store の掲載情報として使えるよう、利用案内とトラブル時の窓口を1ページに整理しています。

問い合わせ先

不具合報告や改善要望は、以下のいずれかの方法で連絡できます。

  • Chrome Web Store のサポート欄からお問い合わせ
  • GitHub Issues: https://github.com/shupopo/fashion-wishlist/issues
  • Issue 作成時は、利用サイト、発生した内容、再現手順、スクリーンショットがあると確認が速くなります。

ヘルプ

  • 保存したい商品ページを開いた状態で、MyDrip を実行してください。
  • 保存済みアイテムは一覧画面から確認し、タグ、メモ、フォルダで整理できます。
  • 同じ商品ページを保存した場合は、重複登録ではなく既存データの更新が行われます。

利用案内

  • MyDrip は、ファッションアイテムを個人用ウィッシュリストとして保存するための Chrome 拡張です。
  • 商品名、画像、価格、URL などの情報は、お使いのブラウザ内の chrome.storage.local に保存されます。
  • 保存データは外部サーバーへ送信されません。詳しくは プライバシーポリシー を確認してください。

Issue 受付先

不具合や要望の管理は GitHub Issues で受け付けています。

  • 新規 Issue 作成: https://github.com/shupopo/fashion-wishlist/issues/new
  • 既存 Issue 一覧: https://github.com/shupopo/fashion-wishlist/issues

FAQ

Q. 保存しても価格や画像が正しく取得できないことがあります。
A. サイト側の構造によっては一部情報を自動取得しづらい場合があります。対象ページの URL と状況を添えて Issue を送ってください。
Q. 保存したデータはどこにありますか。
A. データはブラウザ内の chrome.storage.local に保存されます。サーバーへの同期や外部送信は行いません。
Q. アンインストールするとデータはどうなりますか。
A. Chrome の拡張機能管理から MyDrip を削除すると、ローカルに保存されていたデータも削除されます。
Q. どのサイトで使えますか。
A. 主に商品ページでの保存を想定していますが、各 EC サイトのページ構造によって取得精度は異なります。
サポート窓口まとめ

一般的な問い合わせは Chrome Web Store のサポート欄、不具合報告や改善提案は GitHub Issues の利用が適しています。
