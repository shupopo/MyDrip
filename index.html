Chrome Extension

気になるファッションアイテムを、あとで迷わず見返せる。

MyDrip は、商品ページを開いたままワンクリックでアイテムを保存できる、個人用ファッションウィッシュリストです。画像、価格、URL をまとめて残し、タグやメモ、フォルダで整理できます。

Chrome Web Store サポートを見る プライバシーポリシーを見る
1クリック保存 閲覧中の商品ページから、その場で商品情報を記録できます。
ローカル保存 データはブラウザ内だけに保存され、外部サーバーには送信されません。
整理しやすい フォルダ、タグ、メモで検討中のアイテムを見失いません。

MyDripでできること

スクリーンショットやブックマークを散らかしたままにせず、買うか迷っている服や小物を比較しやすい形で残せます。あとで探し直す手間を減らし、セール待ちやコーデ検討にも使いやすい設計です。

商品情報をまとめて保存

商品名、画像、価格、URL を一覧で確認できるため、候補の比較がしやすくなります。

タグとメモで管理

「春服候補」「セール待ち」など、自分の買い物基準に合わせて整理できます。

重複を防いで更新

同じ商品ページを保存した場合は既存データを更新し、一覧をすっきり保てます。

使い方

1. 商品ページを開く 気になったファッションアイテムのページを表示します。
2. MyDripに保存する 拡張機能を実行すると、ページ情報をもとにアイテムを保存できます。
3. 一覧で見返して整理する 後からフォルダ分けやタグ付け、メモ追記をして検討を続けられます。
